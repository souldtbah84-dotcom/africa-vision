# 🚀 GUIDE — Africa Vision Multiservices
## Mise en ligne en 3 étapes (tout gratuit, sans code)

---

## ✅ ÉTAPE 1 — Créer votre base de données Firebase (10 minutes)

### 1.1 — Créer le projet
1. Allez sur **console.firebase.google.com**
2. Connectez-vous avec votre compte Google
3. Cliquez **"Créer un projet"**
4. Donnez un nom : **africa-vision**
5. Cliquez **Continuer** → désactivez Google Analytics → **Créer le projet**
6. Attendez 1 minute → cliquez **Continuer**

### 1.2 — Enregistrer l'application web
1. Sur la page du projet, cliquez l'icône **< / >** (Web)
2. Donnez un surnom : **africa-vision-app**
3. Cliquez **Enregistrer l'application**
4. Vous voyez un code avec des valeurs. Copiez :
   - La valeur après **apiKey:** (commence par AIzaSy...)
   - La valeur après **projectId:** (ex: africa-vision-12345)
5. Notez ces 2 valeurs sur votre téléphone

### 1.3 — Activer la base de données Firestore
1. Dans le menu gauche, cliquez **Firestore Database**
2. Cliquez **Créer une base de données**
3. Choisissez **Mode test** → cliquez **Suivant**
4. Choisissez un emplacement (laissez par défaut) → **Activer**
5. Attendez 1 minute

---

## ✅ ÉTAPE 2 — Mettre en ligne sur Netlify (5 minutes)

1. Allez sur **netlify.com**
2. Cliquez **Sign Up** → connectez-vous avec Google
3. Sur le tableau de bord, cliquez **"Add new site"**
4. Cliquez **"Deploy manually"**
5. **Dézippez** le fichier africa-vision-netlify.zip
6. **Glissez-déposez le dossier** africa-vision-netlify dans la zone grise
7. Netlify génère votre lien (ex: graceful-sun-123456.netlify.app)

👉 **Optionnel** : Cliquez "Site configuration" → "Change site name" → mettez **africa-vision-gn**

---

## ✅ ÉTAPE 3 — Connecter Firebase (2 minutes)

1. Ouvrez votre lien Netlify dans le navigateur
2. En haut à droite, cliquez le bouton **⚙️ Firebase**
3. Collez votre **API Key** dans le 1er champ
4. Collez votre **Project ID** dans le 2ème champ
5. Cliquez **🔥 Connecter Firebase**
6. Le badge passe de **🟡 Local** à **🟢 En ligne**

---

## ✅ ÉTAPE 4 — Partager avec l'équipe

Envoyez le lien Netlify par WhatsApp à vos collègues.

Chaque collègue fait l'Étape 3 une seule fois avec les mêmes clés.
Ensuite tout le monde voit les mêmes données en temps réel.

💡 **Conseil** : Envoyez les 2 clés (API Key + Project ID) dans un groupe
WhatsApp de l'agence pour que tout le monde puisse les configurer.
